I started in the browser — PHP, Laravel, Vue, Angular — and kept following the
stack down: into the API layer with **NestJS** and **Node**, into infra with
**Docker** and **CI/CD**, and lately all the way down to **Go** CLIs and **C++**
HID code. These days I spend as much time on the *developer experience* of a
project as on the project itself: interactive terminal UIs, one-command release
pipelines, and AI-agent-ready scaffolding.
